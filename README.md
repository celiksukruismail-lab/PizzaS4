Bu proje, HTML ve CSS kullanılarak sıfırdan inşa edilmiş modern ve estetik bir web arayüzüdür.
Amacım, görsel bütünlüğü koruyarak puzzle benzeri düzenleri CSS Grid ve Flexbox ile kusursuz şekilde yeniden üretmek.

✨ Özellikler
Saf HTML & CSS ile yazıldı, hiçbir framework kullanılmadı.

Responsive tasarım: Mobil, tablet ve masaüstünde uyumlu.

Minimalist ve şık görsel düzen.

Öğrenme ve deneme odaklı, adım adım geliştirildi.

🚀 Hedef
Bu proje, hem öğrenme yolculuğumun bir parçası hem de tasarım gücünü kodla birebir yansıtma denemesi.
Kod satırları, görsel bir yapbozun parçaları gibi birleşerek ortaya bütünlüklü bir tasarım çıkarıyor.
